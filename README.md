# Módulos de Ciencias Naturales — Educación Básica Secundaria

**Descripción breve**
Repositorio de módulos didácticos para Ciencias Naturales dirigidos a estudiantes de educación básica secundaria (6°-9°). Los módulos están diseñados para promover aprendizaje activo, contextualizado y basado en evidencias, integrando recursos digitales, guías para el docente y herramientas de evaluación.

## Cómo usar los módulos
1. Explora la carpeta del periodo correspondiente (`/Primer periodo de 2021`).
2. Abre el `MODULO 6` del módulo para ver los módulos por semana en pdf y word.
3. Revisa cada documento para secuencia de actividades, tiempos sugeridos y adaptaciones.
4. Descarga o visualiza los recursos en la carpeta `PDF`.
5. Revisa en cada documento para evaluar los aprendizajes esperados.
